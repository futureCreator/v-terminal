
# v-terminal

## 기본

- 윈도우 환경에서 사용할 터미널 애플리케이션
- 기술스택: Tauri + xterm.js
- 많은 터미널을 띄운다는 가정 하에 성능을 중요하게 생각함
- 디자인은 Apple 휴먼 인터페이스 가이드라인을 철저하게 준수

## 기능

- 상단의 화면분할버튼을 이용해서 바로 분할이 가능함
- 1, 2, 3, 4, 6, 9
- 탭은 폴더를 지정하여 워킹 디렉토리로써 기능
- 탭 내에서 화면 분할 시 해당 폴더로 터미널이 생성
- 프로그램을 종료했다가 다시 켜도 탭과 화면 분할은 저장되어야 함
- 분할된 여러 터미널에 탭 내 전체 패널에 동시에 입력하는 기능
- 테마는 다크만
- 폰트는 JetBrains Mono Nerd Font를 번들 폰트로 사용
